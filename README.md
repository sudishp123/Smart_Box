# AI Projects Repository – Text-to-Speech, Ollama & Camera Codes

This repository contains multiple AI-related projects and experiments, including **Text-to-Speech using NVIDIA Riva Server**, **LLM usage with Ollama**, and a dedicated folder for **camera-related code**.

---

## Text-to-Speech (TTS) Using NVIDIA Riva Server

The following resource was used specifically for implementing **Text-to-Speech AI using the NVIDIA Riva Server on NVIDIA Jetson**:

* JetsonHacks – *Speech AI on NVIDIA Jetson (Text-to-Speech Tutorial)*
  [https://jetsonhacks.com/2023/08/07/speech-ai-on-nvidia-jetson-tutorial/](https://jetsonhacks.com/2023/08/07/speech-ai-on-nvidia-jetson-tutorial/)

This tutorial was used to:

* Set up the NVIDIA Jetson environment for Speech AI
* Configure and run the Riva Server
* Implement Text-to-Speech (TTS) functionality
* Generate audio output from text input

## Ollama (Large Language Model Setup & Usage)
