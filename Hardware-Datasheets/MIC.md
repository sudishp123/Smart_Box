# Mini USB Microphone

## Overview
Compact USB plug-and-play microphone for basic voice input on desktop and laptop systems.

## Technical Specifications

| Parameter            | Value |
|----------------------|-------|
| Interface            | USB 2.0 |
| Microphone Type      | Electret condenser |
| Polar Pattern        | Omnidirectional |
| Driver Requirement   | None (USB Audio Class) |
| Power                | USB-powered |
| OS Compatibility     | Windows, Linux, macOS |
| Intended Use         | Voice calls, speech recognition, basic recording |

## Usage Notes
- Automatically detected as a USB audio input device.
- Select as input source in OS or application audio settings.
- Best performance at close-range voice pickup.
