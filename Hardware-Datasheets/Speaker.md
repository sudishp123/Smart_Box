## Technical Specifications

| Specification        | Details |
|----------------------|---------|
| **Interface**        | USB-A (USB audio class, plug-and-play) |
| **Microphone Type**  | Condenser (electret or similar, USB ADC built in) |
| **Polar Pattern**    | Omnidirectional (typical for basic USB mics) |
| **Power Supply**     | USB bus-powered |
| **Frequency Response** | ~100 Hz – ~16 kHz (typical for small USB mics) |
| **OS Compatibility** | Windows, macOS, Linux (including Raspberry Pi OS) |
| **Driver Requirement** | None (standard USB audio drivers) |
| **Use Cases**        | Voice recording, calls, basic audio input |

## Usage Notes
- Plug into USB port → recognized as audio input device.  
- Select as microphone in OS or application settings.  
- Works with Raspberry Pi and other single-board Linux systems.
